3.3 Funktion

Funktionen sind grundlegende Bausteine in der Programmierung, die einen bestimmten Block von Code zusammenfassen und wiederverwendbar machen. Sie akzeptieren Eingaben (Argumente), führen Aktionen aus und können eine Ausgabe (Rückgabewert) liefern

Funktionen sind in der Informatik viel komplexer als in der Mathematik

Brauchen Schlüsselwort function: 


                                function name(argumente){...}

Wir müssen nur eine Funktion selber schreiben um ein Bild zu machen

Benutzen von bereits definierte Funktionen

3.4 Setup

wird automatisch einmal am Anfang des Programms aufgerufen
legt initialen Aufbau des Codes fest


                            function setup(){
                                // mache erst einmal nichts
                            }

innerhalb dieser Funktion werden wir jetzt Bilder erzeugen

ein Bild erzeugen

1. Schritt: Canvas erstellen


                            createCanvas(width, height);

Mit dieser Funktion wird eine Leinwand (Canvas) für das Zeichnen von Grafiken erstellt. Die Funktion hat zwei Argumente:
width - ist die Breite des Bildes, heigth - ist die Höhe des Bildes

2. Schritt: Hintergrundfarbe auswählen


                            background(color);

Mit dieser Funktion bekommen alle Pixel den gleichen Wert. Die Funktion bekommt als Argument die Werte einer Farbe


                            function setup(){
                                createCanvas(400, 400);
                                background(0);
                            }

Erstellt ein 400 x 400 schwarzes Bild


                            function setup(){
                                createCanvas(700, 300);
                                background(255, 0, 0);
                            }

Erstellt ein 700 x 300 rotes Bild

3.11 Loops

angenommen wir wollen Teile vom Code n-mal wiederholen
aber n ist eine große Zahl oder gar nicht bekannt
n ist z.B. eine zufällige Zahl

Loops automatisieren wiederkehrende Aufgaben, indem sie über eine festgelegte Datenmenge iterieren oder Code eine bestimmte Anzahl von Malen ausführen.

3.11.1 While-Loops

Die while-Schleife führt Code aus, solange eine spezifische Bedingung wahr ist.

Struktur: 

while (Bedingung) { ... }

Beispiel einer while-Schleife:

                            
                                let count = 0;
                                while (count < 5) {
                                    // Code, der so lange wiederholt wird, bis count 5 erreicht
                                    console.log('Count: ' + count);
                                    count += 1;
                                }

3.11.2 For-Loops

Die for-Schleife ist ideal für die Iteration über einen bekannten Wertebereich oder Elemente in einem Array. (später)

Struktur: 

for (Initialisierung; Bedingung; Iteration) { ... }

Beispiel einer for-Schleife:

                            
                                for (let i = 0; i < 5; i += 1) {
                                    // Code, der 5 Mal wiederholt wird
                                    console.log('Iteration ' + i);
                                }

Bsp: Wir generieren eine zufällige Anzahl an Kreisen

Es entsteht ein Raster aus Quadraten

Wir generieren ein Raster aus Quadraten, wobei jedes Quadrat eine 50% Chance hat nicht gemalt zu werden.


                            function setup(){
                                createCanvas(600,600);
                                background(0);

                                let laenge = 50;
                                for(let y=0; y<height; y+=laenge){
                                    for(let x=0; x<width; x+=laenge){
                                        if(random(0,100) >= 50){
                                            rect(x,y,laenge,laenge);
                                        }
                                    }
                                }
                            }
